# 📰 News Classification using LSTM (PyTorch)

A Natural Language Processing (NLP) project built using PyTorch and Long Short-Term Memory (LSTM) networks to classify news articles into different categories.

This project demonstrates a complete NLP text-classification pipeline from raw text to category prediction.

---

# 🚀 Features

✅ Text Tokenization

✅ Vocabulary Creation

✅ Word-to-Index Mapping

✅ Sequence Padding

✅ Word Embeddings

✅ Multi-Class Classification

✅ LSTM Networks

✅ News Category Prediction

---

# 🧠 NLP Pipeline

News Article
